# RickAndMorty-Vue

A Vue 3 + TypeScript web app to browse, filter, and explore Rick and Morty characters.

## Features
- Search characters by name
- Filter by status (Alive, Dead, Unknown)
- Filter by gender
- Sort alphabetically (A–Z / Z–A)
- Click a character to see detailed info in a side panel

## Technologies
- Vue 3
- TypeScript
- Bootstrap

## How to Run
1. Install dependencies:
npm install
2. Start the dev server:
npm run dev
