# BasicCarSharing

A basic car sharing simulation built in Java.

## Features
- View available vehicles of a company
- Rent and return vehicles
- Tracks kilometer stand of each vehicle
- Vehicle categories supported

## Technologies
- Java
- Object-Oriented Programming

## How it Works
- Vehicles have a status (available / rented)
- Renting a vehicle marks it as unavailable
- Returning a vehicle updates the kilometer stand automatically
